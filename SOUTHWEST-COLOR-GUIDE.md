# Southwest Color Palette Implementation Guide

## Color Palette Overview

### Primary Palette (Red Rocks Natural)
- **Cream** `#F5EDD4` - Light backgrounds, cards on dark sections
- **Sky Blue** `#6B9CC3` - Primary accent, links, borders
- **Olive Green** `#6B7037` - Secondary accent, success states
- **Rich Brown** `#5C3639` - Dark backgrounds, deep elements
- **Terracotta** `#D49A7B` - Warm accents, highlights

### Sunset Accents (Dramatic)
- **Sunset Coral** `#E9756D` - Primary CTAs, important buttons
- **Sunset Orange** `#F19456` - Hover states, active elements
- **Sunset Wine** `#8B2E3E` - Featured items, special highlights
- **Sunset Purple** `#5B4260` - Depth, shadows, dark cards
- **Sunset Peach** `#F4B48A` - Soft highlights, subtle accents

### Gold (Premium)
- **Gold** `#F6B800` - Premium elements, special CTAs
- **Gold Light** `#FFCA28` - Gold hover states

## Component Color Mapping

### Navbar
- Background: `Rich Brown #5C3639` with transparency
- Text: `Cream #F5EDD4`
- Logo circles: `Sky Blue #6B9CC3`
- Hover: `Sunset Coral #E9756D`

### Hero Section
- Background: Keep current image + Red Rocks image overlay (lower right)
- Main text: `Cream #F5EDD4`
- Accents: `Sky Blue #6B9CC3`
- CTA Button: `Sunset Coral #E9756D` → hover: `Sunset Orange #F19456`

### About Section
- Background: `Cream #F5EDD4` (light section)
- Text: `Rich Brown #5C3639`
- Cards: `Surface Light #E8DCC5`
- Icons: `Sky Blue #6B9CC3`
- Accents: `Terracotta #D49A7B`

### Shows Section
- Background: `Rich Brown #5C3639` (dark section)
- Text: `Cream #F5EDD4`
- Cards: `Surface Dark #3D3230`
- Featured badge: `Gold #F6B800`
- Borders: `Sky Blue #6B9CC3`

### Shop Section
- Background: `Sunset Purple #5B4260` (dramatic dark)
- Text: `Cream #F5EDD4`
- Cards: `Surface Dark #3D3230`
- Prices: `Gold #F6B800`
- Add to Cart: `Sunset Coral #E9756D`

### Media Section
- Background: `Cream #F5EDD4` (light section)
- Text: `Rich Brown #5C3639`
- Cards: `Surface Light #E8DCC5`
- Play buttons: `Sky Blue #6B9CC3`
- Accents: `Sunset Orange #F19456`

### Contact Section
- Background: `Rich Brown #5C3639` (dark section)
- Form card: `Surface Dark #3D3230`
- Text: `Cream #F5EDD4`
- Input focus: `Sky Blue #6B9CC3`
- Submit button: `Gold #F6B800`

### Footer
- Background: `Sunset Wine #8B2E3E` (deep dramatic)
- Text: `Cream #F5EDD4`
- Links hover: `Sunset Peach #F4B48A`
- Social icons: `Sky Blue #6B9CC3`

## Alternating Pattern
```
Hero (dark) → About (light) → Shows (dark) → Shop (dramatic) → Media (light) → Contact (dark) → Footer (wine)
```

This creates visual rhythm and keeps users engaged!
